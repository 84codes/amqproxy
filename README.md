# AMQProxy

An intelligent AMQP proxy, with AMQP frame rate limiting, connection pooling/reusing etc.

## Installation

```
shards build
bin/amqproxy
```

## Usage

`bin/amqproxy -p PORT -u UPSTREAM:PORT`
